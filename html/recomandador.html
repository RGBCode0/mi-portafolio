🩺 Recomendador de Productos Médicos

Sistema de recomendación web basada en reglas con python

🧠 ¿Qué hace esta aplicación?

Esta aplicación ayuda a los usuarios a recibir recomendaciones de productos médicos (farmacéuticos o naturales) en base al síntoma que presentan. Utiliza una lógica basada en reglas médicas en el backend y muestra el resultado en una interfaz web simple e intuitiva.

⚙️ ¿Cómo funciona?

  • Backend: Desarrollado en Python + Flask. Contiene la lógica que decide qué producto sugerir.
  • Frontend: Desarrollado con HTML, CSS y JavaScript. Permite al usuario seleccionar su síntoma y tipo de tratamiento.
  • Comunicación: El frontend envía una solicitud POST al servidor Flask mediante fetch().
  • Respuesta: El servidor responde con el nombre e imagen del producto recomendado.

📦 Requisitos

  • Python 3.10 o superior
  • Navegador moderno (Chrome, Firefox, etc.)

🚀 Instalación rápida

  1. Clona el repositorio: git clone https://github.com/RaelRGB/recomendadorDeProductosMedicos
  2. Entra a la carpeta backend
  3. Crea un entorno virtual: 
    python -m venv venv
    venv\Scripts\activate  # Windows
  4. Instala las dependencias: 
    pip install -r requirements.txt
  5. Ejecuta el servidor: 
    python app.py
  6. Abre frontend/index.html en tu navegador
