Proyecto: Casa Inteligente

Simulación con Arduino Uno y App Inventor

Descripción General

Este proyecto consiste en una maqueta de una casa inteligente, diseñada e implementada con un microcontrolador Arduino Uno y una aplicación móvil desarrollada con App Inventor. Se simula el acceso a una puerta mediante contraseña y el control de luces vía Bluetooth.

Galería del Proyecto

Objetivos Específicos

  1. Diseñar un sistema de control para el acceso de una puerta que funcione mediante la introducción de una contraseña en un dispositivo móvil.
  2. Implementar el encendido y apagado de luces mediante conexión inalámbrica utilizando un módulo de comunicación Bluetooth.
  3. Integrar y programar los componentes del sistema, como el módulo de conexión y el servomotor para garantizar la interacción entre el dispositivo móvil y el Arduino.

Desarrollo del Proyecto

Se desarrolló una aplicación sencilla en App Inventor que permite al usuario interactuar con el sistema. La aplicación incluye una interfaz para ingresar la contraseña del control de acceso y botones para encender y apagar las luces, asegurando una experiencia intuitiva.

Se programó el Arduino Uno para gestionar el servomotor que controla la cerradura y los pines encargados de las luces. Se hicieron pruebas individuales de cada componente para garantizar su funcionamiento.

Finalmente, se estableció la comunicación con el módulo Bluetooth HC-05, asegurando que las órdenes enviadas desde la aplicación se ejecutaran correctamente.
