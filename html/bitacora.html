📘 Sistema de Bitácora en Java

Diseñado para registrar y consultar el aprendizaje diario. Su propósito es servir como una herramienta de apoyo para la reflexión del progreso, organización del conocimiento y fortalecimiento del proceso de aprendizaje a largo plazo:

🛠️ ¿Qué incluye?

  • ✅ Interfaz gráfica en Java Swing
  • 📂 Base de datos integrada
  • 🖥️ Archivo ejecutable (.exe)
  • ☕ Java Runtime Environment (JRE) incluido

🛠️ ¿Cómo usarlo?

  1. Descarga el archivo ZIP
  2. Extrae el contenido en una carpeta
  3. Ejecuta el archivo sistema-bitacora.exe
